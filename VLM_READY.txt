🔑 下一步：配置 Gemini API Key

  1. 访问：https://makersuite.google.com/app/apikey
  2. 创建或复制 API Key
  3. 在项目根目录创建 .env 文件：
     
     cat > .env << 'ENVEOF'
     GEMINI_API_KEY=your_api_key_here
     ENVEOF

  4. 替换 your_api_key_here 为实际的 API Key
